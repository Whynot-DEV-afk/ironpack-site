# IRONPACK Website

Site officiel d'IRONPACK — Ta moto. Ta route. Ta meute.

Site statique HTML/CSS/JS, déployé sur Vercel.

## Structure

- `index.html` — Site vitrine principal
- `vercel.json` — Configuration Vercel
- `public/` — Assets statiques (favicons, images, etc.)

## Déploiement

Push sur GitHub → Vercel redéploie automatiquement.

🏍️ Made in Europe, for European riders.
